# Healthcare-analysis
The project demonstrates how healthcare data can be used to analyze hospital performance, patient outcomes, readmission risks, and treatment costs.
It highlights real-world analytics workflows used by healthcare analysts and data teams.
# 📊 Project Overview

Healthcare organizations rely heavily on **data analytics** to improve clinical outcomes and operational efficiency.

This project demonstrates a **complete analytics workflow**:

1. **Synthetic healthcare dataset generation**
2. **Data analysis using Python (Pandas)**
3. **SQL analytics queries**
4. **Interactive visualization dashboard**
5. **Exported analytics reports**

The goal is to showcase **real-world data analyst skills used in healthcare analytics.**
